# SHOFY Ecommerce Web application ✅

Shofy is a online web based e-commerce application built using Java EE, Hibernate & Jersey. 🧑‍💻


## Features

- ✅ Jakarta EE
- ✅ Fully Functional Web App
- ✅ Developer tools for maximum productivity
- ✅ Robust error handling techniques
- ✅ Databases & MySQL & Hibernate
- ✅ Testing strategies for robust, reliable code
- ✅ DevOps best practices for agile, collaborative development
- ✅ JavaScript for front-end development
- ✅ JWT an other authentication
- ✅ Google Authentication Implemented
- ✅ Login/Registration systems for user management
- ✅ Jersey Framework for RestFul Web Application Development
